# The weekly update — what leadership actually reads

Leadership reads these on a phone between meetings. The bar:

## Rules

- The ask, if there is one, lives in the first two sentences — never below a status recap.
- One screen total. If it scrolls, cut status, never the risk section.
- Every number carries its delta and window: "42 signups (+8 w/w)", never a bare "42 signups".
- A risk is never sandwiched between two wins and never below the fold; slipped dates state the new date in the same sentence as the slip.
- Green/yellow/red means the same thing every week; a status that changed color says why in one clause.
- Never paste a dashboard screenshot in place of the sentence it should have become.

## Exit

Before sending, read the draft once as the recipient would — phone width, thirty seconds, top to bottom — and check it against each rule; fix what fails, then send.
