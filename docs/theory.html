从零推导 · A Theory of Skills

四原子不是数出来的，
是从循环签名里读出来的

不预设任何既有分类。从"技能物理上是什么"开始：介入面 → 引擎公理 → 循环位点 → 四原子 → 双态 → 正确性 → 判定认识论 → 统计宪章 → 演化。 每一节只使用前面各节已经建立的东西。

穷尽性：结构事实，非概念断言 顺序合法性：六格推导 persona：卸载规则 参考分三层，不混排
00介入面

对象与介入面：只有两条通道

一个基座 agent = 语言引擎 + 工具循环；给定任务，产出一个产物。理论关心的最终量只有一个：产物的质量。一个技能是按需载入的包：一段陈述性文本（SKILL.md）加可选的可执行物。

技能影响 agent 的物理通道恰好有两条：通道 T（文本）——注入引擎所条件化的上下文 token；通道 X（可执行）——向环境提供可调用或可强制运行的程序。没有第三条。

技能的任何主张——"提供知识""保证正确""改进流程"——最终必须兑现为 T 中的一段文本或 X 中的一个程序。后文的穷尽性结论，相对于这个介入面成立（§11）。

01公理

引擎公理：强、不稳、且世界比它大

公理 A（能力）。引擎具备通用推理：读 schema、写代码、分解任务、编排多步工作，携带截止期前的通用知识。这是 Newell 1982 理性原则所要求、迟到四十年才就位的通用符号层引擎。

公理 A′（近似理性）。引擎对自身能力的执行不可靠，且失效系统性、可具名：① 跳过自验；② 过早终止 / 惯性续跑；③ 静默即兴——缺操作时当场编造私有的、未审查的实现且不报告；④ 机械误执行在规模下放大成系统性错误。这是 Simon 有限理性在引擎上的形态。

经验锚 · 与一个巧合

人工策划技能平均 +16.2pp、自动生成技能 −1.3pp（SkillsBench）；过程式技能诱发 skill inertia——提前终止、跳过本会做的验证（CTA）。公理对 A / A′ 恰好是 Newell–Simon 这对合作者：理性原则，由有限理性回火。

公理 B（任务特异性）。任务来自具体领域；领域的事实、判据、操作不完全在引擎内。

A 说引擎强，A′ 说它强而不稳，B 说世界比它大。三条公理共同决定：技能有存在的空间（B）、有不该占的空间（A）、有一类特殊的必须占的空间（A′）。

02缺口空间

从循环签名读出四原子

为什么不从概念出发。一条诱人的推导路径是从"任何目标导向的生产过程"这个概念里分析缺口空间。但概念是开放的，概念分析原则上封不了口——由它得到的穷尽性只能是断言。奠基对象必须是不开放的东西。

正确的奠基对象：循环。基座 agent 的循环不是开放概念，是一段有限的、可检视的具体结构——

Σ · 生成位

Knowledge

缺一个事实——时新的、私有的、领域的。作为引擎条件化的世界模型承重。

Π · 行动位

Capability

缺一个可靠原语——否则引擎静默即兴（A′-3）。作为行动时可调用的操作承重。

φ · 验证位

Judgment

缺一条"对 / 好 / 禁止"的判据。作为对产物施加的检验承重。

γ · 调度位

Control

缺一条非默认的过程规范——门、终止绑定、角色隔离、检查点。作为对续/停/分支的治理承重。

定义（原子的个体化）：一个原子由其内容成为承重的循环位点个体化。这是定义，不是有待证明的主张——原子的种类数等于位点数。

功能层的执照

机制层上一切注入物都是 token，凭什么四位点划分合法？这正是 Newell 1982 的原始论证：一个描述层级合法，当且仅当系统能在该层自洽描述。且这里不止自洽，而且必需——存在只有该划分才能命名的可观察失效：把判据写成事实（错层），内容在生成位被读过一遍、在验证位从未被执行，产物携带缺陷通过一切格式检查。抹平位点区分，就失去了命名这类失效的语言。

归约攻击的定义性消解

"Judgment 是关于好坏的 Knowledge。"句子层面可以改写，但原子由位点个体化：同一句话在生成位是背景知识，在验证位是出口判据——说 Judgment 是 Knowledge，如同说函数的返回类型就是它的参数类型，因为两者都是类型。范畴错误。

"done_when 是判据，所以 Control ⊂ Judgment。"谓词 P 的内容属 φ；"循环在 P 成立前不得终止、且此检查不可跳过"是调度位上的约束，属 γ。Judgment 供谓词，Control 供绑定。谓词无绑定 = 知而不行（A′-1 原样发作）；绑定无谓词 = 空转。二者正交。

Control 的特殊地位：其余三原子填认知缺口（内容缺失，由 B 保证空间）；Control 填执行缺口——内容引擎往往推得出，但不会可靠执行（A′）。即使领域引擎全懂，γ 类介入仍必要。

三角验证 · 四个互不相识的传统

内容侧：注入的句子必属 is / can / ought 三模态之一（休谟的 is/ought，Searle 的适应方向，Ryle 的 knowing-that / knowing-how，von Wright 的道义逻辑；ought 按作用对象二分即 φ 与 γ）。形式侧：MDP 的 (S, A, R, π)、Russell–Norvig 的任务环境、认知科学的 TOTE 单元（1960——test 即 φ、operate 即 Π、exit 即 γ）、SoK 的四元组 S=(C, π, T, R)，各自独立切出同构划分。三角验证不是证明，但四个传统同落一处，不是巧合的形状。

可证伪预测 · 理论主动交出自己的死法

对任何高效技能做逐段消融：每个承重片段（删除后行为 delta 下降的）都应可归入四位点之一；出现"承重但不可归位"的残片，即为反例。该实验可直接排入评估工具的日程。

03顺序

没有 Procedure 原子，顺序有六个合法居所

排除的推导。一个 procedure（固定步骤序列）是一个具体策略——γ 所约束空间中的一个点。策略合成正是公理 A 定义为引擎所有的东西；A′ 指出弱点在策略治理而非合成。所以 γ 位的缺陷是规范性的（过程必须满足什么），不是生成性的（序列长什么样）。供给固定策略是在填一个不存在的缺口，同时替换掉引擎随语境自适应的合成——惯性成本由此而来。这是 Kowalski "Algorithm = Logic + Control" 的技能版。

顺序合法性定理：一条顺序约束属于技能，当且仅当它可还原为某个原子的内容。由 §2 穷尽性，"顺序能住进哪些格"是有限枚举——闭合性是继承来的推论。
格宿主顺序的性质例
1Σ依赖顺序：因果 / 数据依赖的世界事实无法分类尚未挖掘出的模式；含时限
2Σ不可逆顺序：风险事实迁移前必须备份——逆操作不存在
3γ外部强制顺序：道义 / 合规规范采集数据前必须取得同意
4γ认识论顺序：测量 / 理解协议先改动、后测量、再取舍
5φ产物序：产物本身必须呈现的顺序文档章节序、握手先于传数据——出口可检的产物性质，根本不是过程顺序
6Π编译序：焊在原语内部的顺序脚本内部步骤对编排空间不可见——该焊的顺序就编译掉，永远合法

闭合性靠推导而非清单，差别是实质的：清单靠枚举，不知道自己漏了什么；格靠推导，把洞自己找出来——第 5、6 格正是仅靠直觉枚举时最容易漏掉的两格。第 5 格还携带一条设计律——大量被写成 Step 1/2/3 的东西，正确形态是一条 φ（"按此顺序做"改写为"产物须呈现此序"，出口可检，引擎爱怎么达成怎么达成）。第 6 格解释了为什么 scripts/ 内部的顺序从不触发 shuffle test。血统：偏序规划——排序约束只由因果理由生成，其余保持无序。

shuffle test · 一般形式

打乱技能给出的元素，观察哪里失败；每一处失败必须能追溯到六格之一；追溯不到的顺序即焊死的 procedure。推论：新遇到的顺序类型被分类而非追加——教学序 / 认知负荷序落入第 4 格，或编译为渐进披露的物理结构（第 6 格）。有推导的闭合性带预测力，这是它优于清单的证明。

04双态

陈述告知，编译强制

每个原子有两种实现形态，分别落在 §0 的两条通道上（认知科学原型：ACT-R 的知识编译 / 程序化；哲学祖先：Ryle 的 knowing-that / knowing-how）：

原子陈述态（通道 T）编译态（通道 X）
Σ Knowledge正文陈述的事实 / 路由指引查询脚本、数据接口
Π Capability描述一个操作怎么做交付一个具名字段的原语
φ Judgment陈述判据语义验证器
γ Control陈述协议不可绕过的钩子 / CI 门

定理（告知与强制）。陈述态的兑现依赖引擎的执行，继承 A′ 的全部失效；编译态独立于引擎当次状态。推论：凡是必须成立（而不仅是被知道）的正确性，须在 A′ 失效发作的位点具备编译态——

  • 行动位（A′-3/4）→ Capability 必须编译：具名字段原语让错值无槽可落（"使非法状态不可表示"；防错设计同理）。描述"参数依次是……"是陈述态，仍会被位置调用绕过。
  • 验证位（A′-1）→ Judgment 必须在出口编译：验证器检查产物而非过程，对产物如何生成免疫。这是端到端论证（Saltzer / Reed / Clark 1984）的技能形态——任何入口只塑造生产路径，唯有对产物的编译态检验与生产路径无关。出口是保证，入口只降频率。
  • 调度位（A′-1/2）→ Control 的门必须编译为不可跳过。
定理 · 编译的代价

编译买到可强制性，付出可移植性与维护成本（脚本绑定运行时；跨 agent 分发时陈述态是唯一通货）。"机械规则写成散文而非原语"不是无条件缺陷，而是这条权衡曲线上的一个点。评估时给权衡定价，不给形态定罪。

05正文

正文写四件事，然后停笔

  • Σ：世界——概念、类型、关系、结构；含常被遗漏的子类——关于用户的 Σ（意图词表、场景映射）。
  • φ：什么算对——约束、不变量、语义信号。
  • Π 的存在性：声明有哪些原语（"用 scripts/build，它强制具名字段"）——声明存在是 Σ；叙述调用顺序不属于六格，被 §3 排除（该焊的序编译进原语，第 6 格）。
  • γ：过程约束——done_when 谓词 + 绑定、角色隔离、检查点——以约束形式，不以步骤形式。

引擎的份额（怎么走、先后、如何组合）全部删除——不是遗漏，是 §3 的直接要求。路由的推导：载入决策由引擎读元数据做出，所以 description 必须编码缺口签名（"当 X 缺失时"），而不仅是产出描述——路由信息本质上是关于技能自身的 Σ。

persona 卸载规则

"你是 X 领域专家"占据不了任何循环位点：命名不了产物性质（无 φ 可检）、供给不了事实或原语、约束不了过程——只能不可核算地扰动四位点。它在理论自己的测试下结构性失败，不需要禁令：无从出口检验、无从归类缺口、有界步长的改进循环对它无从下手。它不是非法，是不可审计。

规则：persona 是未兑现的原子内容压缩包——作为写作启发合法（先写"像资深审稿人一样"，再逼问自己"审稿人的哪几条判据"），作为出厂内容非法：装运前必须兑现为显式原子内容（"expert"通常在隐式抬高 φ）。
经验锚 · arXiv 2311.10054（EMNLP 2024 Findings）

162 角色 × 4 模型家族 × 2410 道事实题的系统评估：系统提示中加 persona 相比无 persona 对照不提升表现，且效应随角色的性别、类型、领域漂移。同一论文 2023 年初版摘要曾报告一致提升、2024 年扩展后结论翻转——恰是"不可核算扰动"的活体演示。逃生门保持开着：若某条 persona 真在 L2 上稳定产出 delta，理论的回应不是接纳而是翻译——找出它实际扰动的位点，把那内容直接陈述出来。

06正确性

正确 = 恰好填充，失效恰有六种

技能正确 ⇔ 恰好填充其目标缺口：不欠填、不过填、形态正确、有编译态出口。每种失效一个测试：

失效含义测试
无缺口填的是公理 A 已供给的deletion：撤掉技能跑典型任务，仍过 → 不该存在
欠填缺口有残留，引擎仍在即兴improvisation：数被迫即兴的缺口承载物，>0 → 欠填
过填 · 顺序焊入不可归入六格的顺序shuffle（§3 一般形式）
过填 · 成本引擎本已胜任处强加成本inertia-cost：带 / 不带对比，更贵且无增益
错形态该编译处只有陈述（或反之）形态检查：对照 §4 三个必须编译位
无出口缺编译态 φexit：有无对产物的语义检验

deletion 与带/不带对比的方法论身份是 Pearl 意义上的介入：do(有技能) vs do(无技能)，最小的受控实验。

seam 定理 · 复合技能

已声明的 φ 有覆盖边界；边界之外是剩余裁决（机器不可判的语义质量、taste）。γ 必须在 φ 覆盖用尽处停机等人——人拥有的不是 judgment 整体（那会取消 Judgment 原子的意义），而是已声明判据之外的残差。推论：对机器不可判的维度，正确做法是声明不判并路由到 seam，而不是模拟一个判。

07认识论

判定分三层，"独立"意味着去相关

以上测试由谁执行、可信度如何？需要独立推导，因为执行者本身是一个引擎。

L0

机械判定

语法、schema、frontmatter——可靠但只触及形态。入口：降低机械缺陷频率。

L1

阅读预测

LLM 读文本预测效用——无引导 judge 二选一挑更好的技能，准确率 46.4%，不优于掷硬币；仅三个文本信号有效度证据（失效机制编码、可执行具体性、高危黑名单）。须标注为预测。

L2

行为实验 —— 唯一的认证层

带 / 不带技能在留出任务上对比产物。技能是对循环的介入；介入的效应只能由受控的结果对比认证。

推论（测试的层归属）。deletion、improvisation、inertia-cost 的定义含"跑任务"——L2 原生；静态阅读只是它们的 L1 预测版，结论不得冒充 L2。shuffle 的结构半、exit、形态检查文本可查，属 L0/L1 中相对可靠的部分。

定理（相关 judge 的聚合）：中位数压掉采样方差，压不掉共享偏差；同模型同提示的 judge 误差高度相关，第二个 judge 的边际信息趋近于零。这是孔多塞陪审团定理的独立性条件。故"独立"必须定义为去相关：不同模型，或信息隔离。角色隔离是同一定理在演化循环里的形态。
08统计宪章

门里的每一次比较都是统计推断

L2 的测量者（LLM judge + 随机引擎）是噪声源。每一次比较必须声明噪声模型与错误方向：

  1. 效应须越过噪声。任何进入取舍的 delta 都要一条噪声带；带的估计量须与重复次数解耦（分位数或标准差倍数，不用随次数增长的极差）。无带的 delta > 0 放行抖动；无带的 delta ≤ 0 → 杀 误杀低功效集合上的真技能。
  2. 致命项须可复现。逐任务零容忍 × 噪声 judge × 集合规模 = 必然误杀：单任务假回归率 p，n 任务至少一次假致命的概率 1−(1−p)ⁿ（族错误率）。"负迁移致命"限定为在重复运行的中位数上仍成立的回归。
  3. 裁决前先验功效。n 个 pass/fail 任务的 delta 最小刻度 1/n；小于分辨率的效应不可表示。集合无力回答所问问题时，路由到"集合不适格"，绝不折算为技能的失败或成功。四任务集只能出提示性结论。（Neyman–Pearson）
  4. 单调性只对被测集合成立。固定集反复迭代 + 循环可见逐任务结果 = 自适应过拟合（可复用留出集问题，Dwork et al. 2015）。认证"已改进"须依赖循环从未接触的确认切片，或集合轮换 / 封顶轮数。与"不用技能自身历史产出当留出集"是两道独立防线。
  5. 成本属于认证层。§6 的惯性成本以成本为构成要件；若认证层不测 token / 时延，则理论自己定义的一种过填在唯一可靠的判定层上不可测——不融贯。
  6. 两个 delta，两个问题。存在性（vs 无技能）一次判定 + 永久地板；改进性（vs 上一接受版）每轮判定。混同二者的循环在强基座上必然停摆——地板永远是护栏，不是每轮的横杆。
09演化

无门的自编辑不是演化，是受控退化

技能须随经验改进；公理 A′ 决定自编辑不可信（经验锚：自生成技能平均劣化）。总纲是 Campbell 的盲变异-选择保留：变异自由，保留过门。

  • 提议-检验（默认拒绝）：每次编辑过 §8 的改进门才落地。
  • 有界步长：单次改动量封顶（文本学习率），禁止毁灭性单步重写。
  • 负知识持久化：被拒编辑连同分数落差入缓冲，不得重提死路（禁忌搜索的 tabu list）。
  • 批式归纳：对 ≥3 条共享签名的轨迹簇归纳，不对单条失败反应。
  • 对比对为学习单元：失败 × 邻近成功——孤立失败无锚点，全失败池产出最差技能；无邻近成功的失败簇提示缺新能力，路由到抽取而非编辑。
  • 角色隔离：抽取者、执行者、评估者不得互继偏差（§7 去相关的循环形态）。
  • 安全入损失：拒答率、攻击成功率、越界与任务分同门——技能文件是真实攻击面。
  • 抽取的陷阱：把轨迹编译进技能 = 固化一条具体策略 = §3 排除的 procedure。抽取蒸馏模式（哪个位点的缺口反复出现），不是路径——分类前端即问：反复缺席的是事实（→Σ）、被反复即兴的操作（→Π）、反复出现的"好的样子"（→φ）、还是非默认循环（→γ）。

三个演化失效对应三道防线：记忆腐化 → 留出任务保持新鲜、永不取自技能自身历史产出；序贯过拟合 → 批式归纳；错向演化 → 安全门逐编辑运行。

10元技能

理论施加于自身，落出四个元技能

元技能是什么原子构成
evaluate§6 的尺 + §7/§8 的门（内核：出口验证器）Judgment + Capability
write尺反向运行——按 §2–§6 手工作成，含 persona 卸载检查Judgment + Knowledge
seekwrite 由语料驱动 + 存在性门（§9 抽取）Control + Judgment + Capability

各自服从其编码的理论：全部以约束而非步进书写；improve / seek 只声明 done_when + 角色隔离 + 门，任务迭代留给引擎，门协议编译为不可跳过——顺序合法性由 §3 第 4 格给出，不可跳过性由 §4 对 Control 的编译要求给出。

11边界

已知边界：承诺，不是欠账

理论到此完备但不封闭。三条边界，论证到头了，如实划界：

  • 架构索引性。§2 的穷尽性是相对于当前介入面与循环结构的定理，不是永恒真理。若标准演化出新介入通道（技能设置采样参数、路由模型、生成子 agent 拓扑），即出现新参数位，元组须重推。§2 的消融实验是对这条边界的常设审计。
  • 泛化不可解，只可诚实定界。有限留出集上的门只认证到该集合的分布；§8-4 收窄过拟合的窗口，不消灭它。理论的上限是声明边界。
  • 部分 φ 机器不可判。某些语义质量维度不存在编译态——正确的技能行为是声明不判并路由到 seam；"哪些维度属于此类"随模型能力移动，理论只能要求技能显式声明自己的划法。

§12 · 汇流

只在循环的缺陷位点介入，
以公理 A′ 的失效所要求的形态介入，
且仅在 L2 认证介入。

一条轴贯穿始终——留给引擎多少：分类按位点，书写按恰好填充，形态按失效位置，判定按认识论层级，演化按统计宪章。每条规则都是这条轴在一个环节上的投影。
