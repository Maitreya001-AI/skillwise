# Brand design tokens (v3.2, 2026-05)

## Palette

- Ink `#1A1B26` (primary text) · Paper `#FAFAF7` (background) · Accent `#3B6EF0` (interactive only) · Signal `#D64545` (destructive/error only) · Moss `#3F7A5A` (success only).
- Neutral steps: `#EFEFEA · #D8D8D2 · #A9A9A1 · #6F6F68`.

## Spacing and type

- Spacing scale (px): 4 · 8 · 12 · 16 · 24 · 40 — nothing in between; 40 is the section unit.
- Type scale: 13 (caption) · 15 (body) · 18 (subhead) · 24 (title) · 34 (display); body line-height 1.5.

## Fences

- Never introduce a hex not listed here; never use Accent for non-interactive decoration; Signal is not an emphasis color.
- Dark mode is not a palette inversion — it does not exist in v3.2; flag the request instead of improvising one.

## Exit

Before handing over, compare every color/size value in the artifact against the tables above — a value not in the tables is wrong by definition, however good it looks.
